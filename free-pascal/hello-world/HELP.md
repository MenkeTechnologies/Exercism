# Help

## Running the tests

To run the first test, type:

```sh
make
```

To run the second test, type:

```sh
make test=2
```

To run all tests, type:

```sh
make test=all
```

## Submitting your solution

You can submit your solution using the `exercism submit HelloWorld.pas` command.
This command will upload your solution to the Exercism website and print the solution page's URL.

It's possible to submit an incomplete solution which allows you to:

- See how others have completed the exercise
- Request help from a mentor

## Need to get help?

If you'd like help solving the exercise, check the following pages:

- The [Free Pascal track's documentation](https://exercism.org/docs/tracks/free-pascal)
- The [Free Pascal track's programming category on the forum](https://forum.exercism.org/c/programming/free-pascal)
- [Exercism's programming category on the forum](https://forum.exercism.org/c/programming/5)
- The [Frequently Asked Questions](https://exercism.org/docs/using/faqs)

Should those resources not suffice, you could submit your (incomplete) solution to request mentoring.

There are some online resources where you can try to get help:

- [FPC-Pascal users discussions][mailing-list].
- [Forum » Free Pascal » Beginners][forum-beginners].
- [Stack Overflow][stackoverflow]
